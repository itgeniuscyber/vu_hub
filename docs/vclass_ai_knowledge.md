# VClass Knowledge For VU AI Desk

Observed from the VClass student dashboard on 2026-07-06. This note is for
navigation guidance only. Do not store or answer with private student records,
grades, IDs, balances, passwords, or personal profile details.

## Main VClass Sections

- Home: dashboard overview, system updates, announcements, current modules,
  attendance summary, calendar, and upcoming lectures.
- My Modules: active modules, module search, module history, and course material
  access. VClass describes course materials as PDFs, Word documents, and
  presentations organized by week.
- Lectures: lecture list, trimester information, teaching dates, exam dates, and
  a link to the full timetable.
- My Timetable: calendar/timetable area for lectures and schedules.
- Exams & Assessments: routes students to course work, examinations, and
  examination results.
- Financial Statements: financial statement table, PDF download, and tuition
  payment action.
- Tutorials: VClass tutorial videos, including internship guidance.
- VU Elections: election entry point.
- Help & Support: request help from VClass/student support.
- Requests & Applications: resits/retakes, programme change, intake change,
  session change, exemption/credit transfer, and graduation clearance.
- VU Shop: order products and view order history.
- Settings: profile photo upload, password change, and account activity export.

## AI Guidance Rules

- If a student asks where to find course materials, direct them to VClass >
  My Modules and then the relevant module.
- If a student asks about assignments or coursework, direct them to VClass >
  Exams & Assessments > Course Work.
- If a student asks about exams, direct them to VClass > Exams & Assessments >
  Examinations.
- If a student asks for results, direct them to VClass > Exams & Assessments >
  Examination Results.
- If a student asks about resits or retakes, direct them to VClass > Requests &
  Applications > Resits & Retakes.
- If a student asks about programme, intake, or session changes, direct them to
  VClass > Requests & Applications and the matching request type.
- If a student asks about exemptions or credit transfer, direct them to VClass >
  Requests & Applications > Exemption / Credit Transfer.
- If a student asks about graduation clearance, direct them to VClass > Requests
  & Applications > Graduation Clearance.
- If a student asks about tuition, balances, or statements, direct them to VClass
  > Financial Statements and remind them to confirm with Finance for official
  decisions.
- If a student asks for general VClass help, direct them to VClass > Help &
  Support.

## File Reading Workaround

Current VU AI Desk does not read document contents directly. For real file-aware
AI, upload documents to Firebase Storage, extract text server-side, store
summaries/chunks in Firestore, and let the AI retrieve those chunks before
answering.
